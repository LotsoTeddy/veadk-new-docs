---
title: 内置工具
---

本文档介绍智能体开发套件中提供的**内置工具（Built-in Tools）**，包括函数工具与 MCP 工具两大类。  
这些工具是智能体在运行过程中最常调用的外部能力模块，可帮助开发者快速构建具备计算、生成、检索、通信等能力的复合型 Agent。

---

## 使用内置工具构建复杂应用

本文档建议通过内置工具组合，构建具有“感知—推理—行动”闭环能力的复杂应用。  
以下为可供新手参考的示例思路（无需立即实现，可在后续章节中补充代码逻辑）：

| 应用场景 | 工具组合 | 功能说明 |
|-----------|-----------|-----------|
| **智能报告生成助手** | `image_generation` + `Sandbox` + `Lark` | 从文本生成图表与摘要，执行计算后推送报告至 Lark 群组。 |
| **自动化客服回复系统** | `Lark` + `LAS` + 函数工具（文本摘要） | 监听客户消息并自动生成回应，必要时调用 LAS 触发后续工单流程。 |
| **多模态内容创作系统** | `image_generation` + 其他函数工具 | 接收文本指令，自动生成配图与标题，支持内容迭代。 |
| **安全测试 Agent** | `Sandbox` + 函数工具（检测类） | 在隔离环境中执行测试脚本，并生成自动化报告。 |
